# CodeRabbit Review Commands

## Standard flow

1. Request review on active PR:

```text
@coderabbitai review
```

1. If CodeRabbit is paused:

```text
@coderabbitai resume
```

1. Submit/continue review:

```text
@coderabbitai pause
@coderabbitai resume
```

## Operational notes

- Use review comment resolution tools as they are required by repository policy.
- If a comment is actionable, implement and reply with short justification.
- Do not dismiss reviews manually unless explicitly requested by the user.
